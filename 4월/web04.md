## Spring Core 모듈
- 스프링은 싱글톤 패턴을 이용함
- 클래스 객체를 1개로 제한하는 방법
- static을 잘 이해하고 있으면 구현 가능함
- java singleton pattern
- 죽었다 깨어나도 단 하나의 객체만 생성됨
### 프로그래밍 디자인패턴
- 자료구조랑 비슷함
- 언젠가는 공부를 하긴 해야함
- MVC, 파사드, 옵저버, 브릿지 ...

### 객체 생성
- 생성자 주입을 사용하자
- 생성자에 @Autowired를 붙이면 생성자 주입
- 세터에 @Autowired를 붙이면 setter 주입
- 스프링은 필드에서 @Autowired도 붙일 수 있음 (테스트할 때 사용)
- 필드 주입은 테스트할 때만 쓰자, 실무에서 불리하다
